  1. варіант помилки нема, виведе текст, можливо така умова
  2. варіант вірний, дія множення виведе число
  3. варіант викличе помилку, ++ не додається числу
  4. варіант вірний, спочатку виконається дія 0 потім +1
  5. варіант вірний, поверне перше true з справа, так як !false і !'' це true
  6. варіант вірний, поверне true

Умовний оператор

  1. варіант - верне ложь, так як 1 не дорівнює рядку 1
  2. варіант верне істина, так як 1 не строго дорівнює рядку 1
  3. варіант верне істина так як тут два true виведе true і в результаті істина
  4. варіант - 0 це false, отже не ложь, " " це true отже виведе істина
